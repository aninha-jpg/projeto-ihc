Este projeto foi desenvolvido como atividade prática da disciplina de Interação Humano Computador, do curso Análise e Desenvolvimentos de Sistemas da PUCPR.
Com o objetivo principal de criar personas utilitárias do sistema e suas storytellings, analisando os perfis e comportamentos para auxiliar no design centrado no usuário(UX).
Também realizado uma avaliação heurística, visando identificar problemas e propor melhorias.

Status: Projeto concluído e utilizado somente para fins acadêmicos.
Data: 28/09/2025.
